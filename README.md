# AxisRank

AxisRank（アクシスランク）は、2つの評価軸（X軸・Y軸）を用いて要素を2次元マップ上に視覚的に配置し、その座標から総合スコアとランキングを自動算出する直感的な評価・分析ツールです。

## 🚀 主な機能

- **2D空間マッピング**: 評価したい要素（アイテム）を、ドラッグ＆ドロップで直感的に2D座標上へ配置できます。
- **柔軟な軸設定（Axes Settings）**: X軸・Y軸それぞれの名称、ラベル（低い/高いなど）、および重要度（ウェイト）を自由にカスタマイズ可能です。
- **リアルタイムランキング計算**: 要素をマップ上で動かすたびに、指定した評価方式（加重和など）に基づき、即座に総合スコアと順位が計算・更新されます。
- **2つの洗練されたテーマデザイン**:
  - **Dark Mode (FUI)**: 漆黒の背景にシアンブルーのネオンドットや発光エフェクトが映える、テクニカルで硬派なSF風インターフェース。
  - **Light Mode (Retro Future)**: クリーム色の背景に煙色のグリッド、そしてオレンジのアクセントカラーを配した、ノスタルジックなレトロフューチャーデザイン。
- **インタラクティブなマップ操作**: パン（画面移動）やズーム機能に対応しており、要素が多くても快適に操作できます。
- **ローカル自動保存機能**: プロジェクトや要素のデータはすべてブラウザのローカルストレージに自動で保存されるため、リロードしても作業を再開できます。

## 🛠️ 技術スタック

- **フレームワーク/ライブラリ**: React (v19), TypeScript
- **スタイリング**: Vanilla CSS
  - 外部のCSSフレームワーク（Tailwind等）には依存せず、CSS Variables (Custom Properties) を駆使して高度なテーマ切り替え（Dark/Light）を実現しています。
- **ビルドツール**: Vite
- **グラフィックス**: 素のSVG（`<svg>`）を利用して、軽量かつインタラクティブなマップ（グリッド、レーダー円、グラデーション描画、結線など）をフルスクラッチで構築しています。

## 📂 プロジェクト構造

```text
src/
├── components/         # Reactコンポーネント群
│   ├── MapView.tsx     # メインとなる2Dインタラクティブマップ（SVG描画・ドラッグ＆ドロップ）
│   ├── LeftSidebar.tsx # プロジェクト一覧と未配置要素の管理
│   ├── RightSidebar.tsx# リアルタイムのランキング表示パネル
│   ├── Header.tsx      # タイトル、軸設定、テーマ切り替えトグル
│   └── modals/         # 軸設定などのモーダルUI
├── core/               # ドメインロジック・アルゴリズム
│   ├── ranking.ts      # 座標からのスコア計算・順位付けロジック
│   ├── storage.ts      # ローカルストレージとの同期処理
│   └── types.ts        # プロジェクト全体の型定義
├── state/              # グローバル状態管理
│   ├── context.tsx     # React Context API と useReducer を用いた状態管理
│   └── hooks.ts        # カスタムフック
└── styles/             # CSS設計
    ├── index.css       # デザイン変数（テーマカラー、フォント、余白など）
    ├── layout.css      # 大枠のレイアウト（CSS Grid）とマップ背景
    └── components.css  # 各種UIコンポーネント（ボタン、チップ、モーダルなど）の詳細スタイル
```

## 🏃‍♂️ ローカルでの起動方法

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. 開発サーバーの起動:
   ```bash
   npm run dev
   ```

3. ブラウザで `http://localhost:5173` にアクセスしてください。
